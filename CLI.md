# CLI

- Command Line Interface

- 일명 가짜 명령 프롬프트

- 터미널

  - 인터페이스란?

    서로 다른 두 환경을 만나게 해주는 중간체

-  touch (파일명).(확장자)  >> 파일 생성

- touch .hidden.txt  >> 숨겨진 파일 생성

- ls -a >> 숨겨진 파일 보여주기

- `mv` bbb.txt CLI/   >>bbb 텍스트 파일을 CLI로 옮겨라는 의미

- rm bbb.txt   > > bbb 텍스트 지우기

- ls  리스트 보여주기		

- cd ../
  - 상위 폴더(디렉토리)로 이동
  - cd만 입력하면 홈폴더로 이동
- ~ 
  - 루트: 최상단 폴더

- mkdir git-practice
  - 폴더 만들기
- code .
  - 비주얼스튜디오 코드 프로그램으로 열기
